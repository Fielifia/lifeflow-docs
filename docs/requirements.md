# Requirements

This document contains the functional and non-functional requirements that defined the scope of the LifeFlow Fitness MVP.

Requirements were used throughout development to guide implementation, testing, and validation.

---

## Functional Requirements
| ID     | Title                   | Summary                                    | Status        |
| ------ | ----------------------- | ------------------------------------------ | ------------- |
| REQ-1  | User Registration       | Allow users to create accounts             | ✅ Implemented |
| REQ-2  | User Login & Session    | Enable authentication and session handling | ✅ Implemented |
| REQ-3  | Log Workout Sessions    | Users can create and complete workouts     | ✅ Implemented |
| REQ-4  | Personal Best Detection | Detect and highlight personal bests        | ✅ Implemented |
| REQ-5  | Workout Templates       | Create and manage templates                | ✅ Implemented |
| REQ-6  | Edit Workouts/Templates | Modify existing workouts and templates     | ✅ Implemented |
| REQ-7  | Calendar Planning       | Schedule future workouts                   | ⏳ Planned     |
| REQ-8  | Statistics Overview     | View workout statistics                    | ✅ Implemented |
| REQ-9  | Notes                   | Add notes to workouts and exercises        | ✅ Implemented |
| REQ-10 | Exercise Library        | Browse, search and filter exercises        | ✅ Implemented |
